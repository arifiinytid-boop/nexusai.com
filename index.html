⬡ NEXUS AI Plugin — Install Guide

Method 1: Toolbox

1. Open Roblox Studio → View → Toolbox

2. Search: NEXUS AI Studio Plugin

3. Click Install


Method 2: Manual (.rbxmx)

1. Download nexus-plugin.rbxmx

2. Copy to: %LocalAppData%\Roblox\Plugins\

3. Restart Studio


⚠ Required Setup

• In Studio: File → Settings → Security → Allow HTTP Requests ✓

• Run bridge: node bridge.js

• Click NEXUS AI → CONNECT in Studio toolbar
